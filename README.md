 🌱 Smart Waste Management System (Group Project)

A full-stack *Smart Waste Management System* developed as a *group project* using the MERN Stack. The application streamlines waste collection, complaint management, and monitoring through role-based access for Admin, Collector, and Citizen.

## 👥 Team Project

This project was developed collaboratively by a team of students as part of our MCA curriculum.

### My Contributions
- Developed frontend pages using React.js.
- Implemented JWT-based user authentication and role-based access control.
- Designed and integrated REST APIs with the frontend.
- Assisted in MongoDB database design and testing.
- Collaborated with team members using Git and GitHub.

## 🚀 Key Features
- User Authentication (JWT)
- Role-Based Access (Admin, Collector, Citizen)
- Waste Complaint Registration & Tracking
- Complaint Assignment & Status Updates
- Dashboard & Reports
- Responsive User Interface

## 🛠️ Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- JWT
- Git & GitHub
